Crops Nutrients Prediction - Project at ISRO
Project at IIM Bangalore



The project segments green crops from drone images into individual plots for detailed analysis. Using Python libraries like Rasterio and Geopandas, it calculates the (NDVI), applies a threshold to create a vegetation mask, and uses contours to detect boundaries, which are converted into polygons. Further analysis, such as calculating the mean NDVI, assesses crop health and predicts yield.
